

# Required

# Installation

```

# JAVA_HOME=/usr/lib/jvm/java-1.7.0-openjdk-amd64/ mvn clean install -e -DskipTests -Dspark.version=1.1.0 -Dhadoop.version=2.3.0-cdh5.0.0
```
